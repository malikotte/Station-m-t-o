# Station-météo

![alt text](https://www.francetvinfo.fr/image/75wht3mq6-ec36/840/472/23387013.jpg)

Objet connecté : Permet de mesurer température / Humidité de la pièce + Activation via google home d'un servomoteur

Matériels : 
- ESP8266
- AM2302
- Servomoteur
- Google Home

Technologies : 
- API
- MQTT
- Node-red
